Introduction
------------

This project contains the code and configuration needed to integrate
`Trino <https://trinodb.io/>`_ with `Ambari <https://ambari.apache.org/>`_.
Adding the Trino service to Ambari allows:

1. Installing and deploying Trino on a cluster from the Ambari UI.
2. Changing Trino configuration options via the Ambari UI.
